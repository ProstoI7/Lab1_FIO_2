# Lab1_FIO
## Описание проекта
Данный проект представляет собой консольное приложение на языке C#, которое анализирует геометрические свойства треугольника на основе длин его сторон.

### Основной функционал:
1. **Определение типа треугольника:** «равносторонний», «равнобедренный», «разносторонний», «не треугольник», «» (пустая строка для нечисловых входных данных).
2. **Расчет координат:** вычисление координат вершин треугольника для отрисовки в поле 100x100 пикселей.
3. **Логирование:** - Все операции записываются в консоль и в текстовый файл `log.txt`.
   - Успешные операции подсвечиваются (в консоли) и содержат результаты расчетов.
   - Ошибки (некорректный ввод, нечисловые данные) записываются с указанием причины.
